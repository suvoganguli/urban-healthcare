# Urban Healthcare Inequality Mapping

This project analyzes spatial inequality in access to healthcare facilities within a city.

Instead of focusing on building new infrastructure, the goal is to identify mismatches between population distribution and existing healthcare access.

## Core Components

- High-resolution population data (HRSL)
- Healthcare facility locations (OpenStreetMap)
- Accessibility modeling (distance / travel time)
- Gap metric to identify underserved regions

## Output

- Accessibility map
- Population density map
- Healthcare inequality (gap) map
- Hotspot identification

## Philosophy

Healthcare access is not uniformly distributed.  
The issue is not absence of hospitals, but spatial mismatch between population demand and accessibility.
